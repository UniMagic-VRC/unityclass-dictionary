授業で用いるスライドの左上には、重要度や難易度を示すアイコンが付与されています。

| 種別   | 説明                                                         |
| ------ | ------------------------------------------------------------ |
| (なし) | 通常のスライド                                               |
| ⭐     | 重要なスライド(これだけは覚えて帰ってね！)                   |
| 🔥     | 難しいスライド(授業としては取り上げずに読み飛ばす可能性あり) |
